# PrepAI — AI Mock Interview Assistant
### by Clonigue

PrepAI is a desktop-based AI-powered mock interview assistant that helps 
you practice technical interviews with real-time voice recognition, 
facial expression analysis, and detailed performance reports.

## Features
- AI-generated domain-specific questions
- Voice recognition for answers
- Facial expression analysis during interview
- Detailed PDF performance report
- Session history tracking

## Domains Supported
- Python
- DBMS & SQL
- Operating Systems
- JavaScript

## Tech Stack
- Python 3.11
- CustomTkinter (GUI)
- Whisper (Voice Recognition)
- DeepFace + FER (Facial Analysis)
- Gemini API (Question Generation + Scoring)
- ReportLab (PDF Reports)
